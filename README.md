# 葛飾の波 — 創発

### Katsushika Wave: Emergence

---

二千の粒子は、波のことを何も知らない。

それぞれが知っているのは、自分の位置と、そこに吹く微かな風だけ。三角関数の重ね合わせが作る力場——それは海底の地形のようなもの——に押され、引かれ、ただ漂う。

それなのに、波が立つ。

潮が満ちる。泡が生まれる。北斎が見たであろうあの海が、コードの中から自ずと現れる。

これが **創発（emergence）** である。部分の総和を超えた何かが、全体として立ち上がる瞬間。脳が世界を「見る」のも同じ原理だ。網膜の一つ一つの細胞は光の点しか知らない。それなのに、私たちは夕焼けに心を動かされる。

三つの呼吸がある。

| | |
|---|---|

| **記憶（Memory）** | 過去の軌跡がどれだけ残るか。上げれば水墨画の筆致が現れ、下げれば粒子は今この瞬間だけを生きる。 |
| **再生（Rebirth）** | すべてを無に還し、新しい海を生む。同じ海は二度と現れない。 |

## コードは詩である

変数名を読んでほしい。

```
life, maxLife       — 生と死
breathe             — 呼吸
phase               — 位相、あるいは人生の段階
foam                — 波頭に散る白い泡
```

このプログラムには「波を描け」という命令は一行もない。あるのは、粒子の `life` と `breathe` と、場に吹く風だけ。波は命令されて生まれるのではない。生きることの結果として、現れる。

## 技術的な注記

- **依存関係**: なし。純粋なHTML + Canvas + JavaScript。
- **粒子数**: 2000（端末の性能に応じて調整可能）
- **描画**: 各粒子は独立に力場を参照し、速度を更新する。全体の協調は力場の共有のみによって生じる。
- **黄金比**: キャンバスの縦横比は `0.618`（φの逆数）。自然が好む比率。
